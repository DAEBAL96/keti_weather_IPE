초단기 실황 -> 1일간의 데이터만 제공

0000 ~ 0059 사이의 값을 조회시 0000 base time의 초단기 실황 값을 불러온다.
-> 어떻게 값을 넣어줄지 잘 봐야할듯? -> time_stamp랑 format 맞춰서 넣어주면 되겠네.

PTY -> 강수 형태
	0 없음, 1 비, 2 비/눈, 3 눈, 5 빗방울, 6 빗방울, 7 눈날림
REH -> 습도

RN1 -> 1시간 강수량

T1H -> 기온

uuu -> 풍속(동서성분)

VVV -> 풍속(남북성분)

WAV -> 파고

VEC -> 풍향

WSD -> 풍속
-----------------------------------------
사용할 요소 
T1H(기온 -> 값 그대로 가져감), PTY(비 또는 흐림)
